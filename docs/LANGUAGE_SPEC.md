# GS++ (Gian Stagno Plus Plus) — Language Reference

This document is a short pointer. For the **full specification**, see:

- **[docs/GSPP_SPEC.md](docs/GSPP_SPEC.md)** — Complete language spec (types, syntax, stdlib, safety, toolchain)

## Quick reference

- **Extension:** `.gs`
- **Compiler:** `gsc`
- **Keywords:** `var`, `let`, `func`, `def`, `class`, `struct`, `if`, `else`, `while`, `for`, `return`, `import`, `asm`, `unsafe`, …
- **Types:** `int`, `float`, `bool`, and user `struct`/`class`
- **Style:** Python-like readability (`def`/`let`/`class`) with C++-like performance (compiled native code)
